# Ateliê da Casa Museu Ema Klabin

## Plugin WordPress

1. rodar `npm run bundle` para gerar o .zip do plugin
2. fazer o upload do `atelie-cmek.zip` no wordpress
   
### Shortcode WordPress

- usar a tag de shortcode `[atelie-cmek]` onde deseja que o jogo apareça
